Parameter Guide — edit the sliders above without touching the code

Transmission Rate — base probability per tick that an infectious agent spreads to a nearby healthy agent. Higher = faster spread.

Poor Zone Multiplier — how much more likely poor-zone agents are to get infected (overcrowding, no clean water). Default 2.5×.

Hospital Debt Barrier — % chance a poor uninsured patient gets turned away from private hospitals. 0 = open access, 100 = total exclusion.

Surge Cost — multiplier on private hospital costs during crisis. Affects how many middle-class agents can afford care.

Vaccine Coverage — % of rich/poor population that starts vaccinated (immune). Reflects pre-existing inequality in healthcare access.

To add new contamination zones or hospitals, edit the WATER_SOURCES and HOSPITALS arrays in the script below.
